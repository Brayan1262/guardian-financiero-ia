# Notas de Desarrollo y Planificación

El proyecto se desarrollará de forma progresiva, dividiendo el trabajo en los siguientes módulos:

* **Módulo 1:** estructura base del monorepo.
* **Módulo 2:** backend Java Spring Boot base.
* **Módulo 3:** configuración de PostgreSQL.
* **Módulo 4:** autenticación JWT y roles.
* **Módulo 5:** gestión de clientes.
* **Módulo 6:** gestión de transacciones.
* **Módulo 7:** motor antifraude basado en reglas.
* **Módulo 8:** generación y gestión de alertas.
* **Módulo 9:** dashboard Angular.
* **Módulo 10:** microservicio IA con FastAPI.
* **Módulo 11:** integración Java Spring Boot con FastAPI.
* **Módulo 12:** auditoría de acciones.
* **Módulo 13:** pruebas unitarias.
* **Módulo 14:** Docker Compose.
* **Módulo 15:** documentación final y preparación para GitHub.
